# 🔥 CCLNet: UAV-based Forest Fire Detection Dataset (TF-11K)

## 📌 Dataset Overview
- ​**Total Images**: 11,139 (9,139 self-collected + 2,000 from FLAME)
- ​**Annotations**: PASCAL VOC format (XML files)
